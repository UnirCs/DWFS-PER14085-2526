# API REST – Calculadora Online (con memoria)

## Objetivo
Diseñar una API REST para una calculadora online que soporte operaciones matemáticas y mantenga un historial consultable.  
Cada operación ejecutada se almacena y queda disponible mediante un **ID de operación**.
