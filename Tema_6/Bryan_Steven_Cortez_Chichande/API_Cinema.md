## API de un sistema de reserva de butacas de cine

## Objetivo
Diseñar una **API REST** para un sistema de **reserva de butacas de cine** que permita administrar **películas**, **salas** y **usuarios**, además de gestionar el ciclo completo de una **reserva** (crear, modificar y cancelar) y registrar el **pago** asociado. La API debe utilizar **URIs basadas en recursos**, manejar correctamente **códigos de respuesta HTTP**, y permitir identificar y operar sobre entidades mediante sus **IDs**.
